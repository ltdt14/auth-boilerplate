## Commands

Do everystep for production at once

```
npm run production
```

Compile API doc

```
npm run apidoc
```

Compile API doc (HTML + Markdown)

```
npm run apidocmd
```

Compile API doc

```
grunt jsdoc2md
```

Run Coverage

```
npm run coverage
```

Run Tests

```
npm test
```

Add table of contents to readmes

```
grunt markdown_toc
```

Linting

```
grunt eslint
```
